# Social Readiness Docs Design

Date: 2026-03-23

## Goal

Close the narrow social-readiness gap in the active Romance Unzipped episode by adding the missing documentation artifacts that current issues explicitly call for: hashtag strategy and post validation checklist.

## Scope

Create two episode-scoped files in the current episode's `assets/social/` directory:

1. `hashtag-strategy.md`
2. `post-validation-checklist.md`

These files should support the existing Instagram, TikTok, and Facebook draft assets without changing the voice or structure of those draft posts.

## Content Design

### Hashtag Strategy

The strategy file should be practical rather than generic. It should cover:

- a small primary tag set that should anchor every post
- platform-specific supporting tags for Instagram, TikTok, and Facebook
- a rotation rule to avoid repetitive or spam-like tag blocks
- a short "avoid" list for overbroad, misleading, or clutter-heavy tags

### Post Validation Checklist

The checklist file should be release-oriented. It should cover:

- caption and quote accuracy
- episode/book metadata correctness
- CTA/link consistency
- subtitle and clip sanity checks
- platform-fit checks before manual posting
- final approval and no-publish guardrail

## Success Criteria

- `ROM-413` has a concrete artifact to point at
- `ROM-416` has a concrete artifact to point at
- the new docs fit the existing Romance Unzipped tone and workflow
- no external posting or platform mutation is performed
